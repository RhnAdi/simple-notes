# Simple Notes with React and Firebase

## Made with :

1. React
2. Firebase
3. Redux
4. Redux-Thunk
5. React-Hooks
6. Sass

## Get Started :

1. Clone Repository
   > git clone https://github.com/RhnAdi/simple-notes
2. Install Node Modules
   > npm install
3. Start
   > npm start
